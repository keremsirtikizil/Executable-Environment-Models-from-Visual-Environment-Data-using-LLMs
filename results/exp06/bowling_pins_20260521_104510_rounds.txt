```pseudocode
STATE VARIABLES:
  sweep_arm_state: (retracted, lowering, sweeping)
  pins: LIST of Pin objects

RULES:
  IF event IS clear_pins AND sweep_arm_state IS retracted:
    THEN SET sweep_arm_state to lowering

  IF sweep_arm_state IS lowering AND sweep_arm.y_position <= lane_surface.y_position:
    THEN SET sweep_arm_state to sweeping

  IF sweep_arm_state IS sweeping:
    THEN MOVE sweep_arm forward
    FOR EACH pin in pins:
      IF sweep_arm collides_with pin:
        THEN REMOVE pin from lane
```

Round 2  —  frames [3, 4, 5]
==================================================

```pseudocode
STATE VARIABLES:
  ball: Ball object with properties (position, velocity)
  pins: LIST of Pin objects, each with properties (id, position, velocity, state: {standing, falling, fallen})
  game_state: (ball_rolling, pins_interacting, frame_over)

RULES:
  IF game_state IS ball_rolling AND ball collides_with a pin:
    THEN SET game_state to pins_interacting
    TRANSFER momentum from ball to the hit pin
    SET hit_pin.state to falling

  IF game_state IS pins_interacting:
    FOR EACH pin_A in pins:
      IF pin_A.state IS falling:
        UPDATE pin_A.position and pin_A.velocity
        FOR EACH pin_B in pins where pin_A IS NOT pin_B:
          IF pin_A collides_with pin_B AND pin_B.state IS standing:
            THEN TRANSFER momentum from pin_A to pin_B
            SET pin_B.state to falling

  // Specific interaction observed
  IF pin with id=6 collides_with pin with id=10:
    THEN APPLY strong lateral force to pin with id=10

  IF game_state IS pins_interacting AND all pins have stopped moving:
    THEN SET game_state to frame_over
```

Round 3  —  frames [6, 7, 8]
==================================================

```pseudocode
STATE VARIABLES:
  game_state: (ready_for_roll, ball_rolling, pins_interacting, frame_over, reset_sequence)
  ball: Ball object with properties (position, velocity, color)
  pins: LIST of Pin objects, each with properties (id, position, velocity, state: {standing, falling, fallen})
  sweep_arm_active: BOOLEAN
  pin_setter_active: BOOLEAN

RULES:
  IF game_state IS ball_rolling AND ball collides_with a pin:
    THEN SET game_state to pins_interacting
    TRANSFER momentum from ball to the hit pin
    SET hit_pin.state to falling

  IF game_state IS pins_interacting:
    FOR EACH pin_A in pins:
      IF pin_A.state IS falling:
        UPDATE pin_A.position and pin_A.velocity
        FOR EACH pin_B in pins where pin_A IS NOT pin_B:
          IF pin_A collides_with pin_B AND pin_B.state IS standing:
            THEN TRANSFER momentum from pin_A to pin_B
            SET pin_B.state to falling
    IF all pins and ball have stopped moving:
      THEN SET game_state to frame_over

  IF game_state IS frame_over:
    THEN SET game_state to reset_sequence

  IF game_state IS reset_sequence:
    // This block represents the unseen clearing and setting mechanism between rolls
    SET sweep_arm_active to TRUE
    REMOVE all pins from lane
    SET sweep_arm_active to FALSE
    SET pin_setter_active to TRUE
    RESET pins to a full rack of 10 standing pins
    SET pin_setter_active to FALSE
    SET game_state to ready_for_roll

  IF game_state IS ready_for_roll AND a ball is detected entering the lane:
    THEN SET game_state to ball_rolling
    INITIALIZE ball with new properties (e.g., color, velocity)
```

Round 4  —  frames [9, 10, 11]
==================================================

```pseudocode
STATE VARIABLES:
  game_state: (ready_for_roll, ball_rolling, pins_interacting, frame_over, reset_sequence)
  ball: Ball object with properties (position, velocity)
  pins: LIST of Pin objects, each with properties (id, position, velocity, state: {standing, falling, fallen})
  sweep_arm: SweepArm object with properties (state: {retracted, lowering, sweeping, retracting})
  pinsetter_deck: PinsetterDeck object with properties (state: {raised, lowering, lowered, raising})

RULES:
  IF game_state IS reset_sequence AND pinsetter_deck.state IS lowered:
    THEN SET pinsetter_deck.state to raising
    CREATE new set of 10 standing pins at default positions

  IF pinsetter_deck.state IS raising AND pinsetter_deck is fully raised:
    THEN SET game_state to ready_for_roll

  IF game_state IS ready_for_roll AND a ball is detected entering the lane:
    THEN SET game_state to ball_rolling
    INITIALIZE ball with new properties

  IF game_state IS ball_rolling AND ball collides_with a pin:
    THEN SET game_state to pins_interacting
    TRANSFER momentum from ball to the hit pin
    SET hit_pin.state to falling

  IF game_state IS pins_interacting:
    FOR EACH pin_A in pins:
      IF pin_A.state IS falling:
        UPDATE pin_A.position and pin_A.velocity
        FOR EACH pin_B in pins where pin_A IS NOT pin_B:
          IF pin_A collides_with pin_B AND pin_B.state IS standing:
            THEN TRANSFER momentum from pin_A to pin_B
            SET pin_B.state to falling
    IF all pins and ball have stopped moving:
      THEN SET game_state to frame_over

  IF game_state IS frame_over:
    THEN SET game_state to reset_sequence
    SET sweep_arm.state to lowering

  IF sweep_arm.state IS lowering AND sweep_arm reaches lane:
    THEN SET sweep_arm.state to sweeping

  IF sweep_arm.state IS sweeping:
    MOVE sweep_arm forward
    FOR EACH pin in pins:
      IF sweep_arm collides_with pin:
        THEN REMOVE pin from lane
    IF sweep_arm reaches end of pin deck:
      THEN SET sweep_arm.state to retracting

  IF sweep_arm.state IS retracting AND sweep_arm is fully retracted:
    THEN SET pinsetter_deck.state to lowering
```

```pseudocode
STATE VARIABLES:
  game_state: (ready_for_roll, ball_rolling, pins_interacting, frame_over, reset_sequence)
  initial_hit_type: (none, pocket, high_pocket, brooklyn)
  ball: Ball object with properties (position, velocity)
  pins: LIST of Pin objects, each with properties (id, position, velocity, state: {standing, falling, fallen})
  sweep_arm: SweepArm object with properties (state: {retracted, lowering, sweeping, retracting})
  pinsetter_deck: PinsetterDeck object with properties (state: {raised, lowering, lowered, raising})

  IF game_state IS ready_for_roll AND a ball is detected entering the lane:
    THEN SET game_state to ball_rolling

  IF game_state IS ball_rolling AND ball collides_with pin_1:
    THEN SET game_state to pins_interacting
    IF ball.impact_point is between pin_1 and pin_2:
      THEN SET initial_hit_type to high_pocket
    TRANSFER momentum from ball to pin_1
    TRANSFER momentum from ball to pin_2

  IF game_state IS pins_interacting:
    UPDATE ball.position and ball.velocity
    FOR EACH pin in pins:
      IF pin.state IS falling:
        UPDATE pin.position and pin.velocity

    IF initial_hit_type IS high_pocket:
      IF pin with id=2 collides_with pin with id=4 AND pin with id=4.state IS standing:
        THEN SET pin with id=4.state to falling
        APPLY velocity to pin with id=4 towards pin with id=7
      IF pin with id=4 collides_with pin with id=7 AND pin with id=7.state IS standing:
        THEN SET pin with id=7.state to falling

    FOR EACH pin_A in pins:
      IF pin_A.state IS falling:
        FOR EACH pin_B in pins where pin_A IS NOT pin_B:
          IF pin_A collides_with pin_B AND pin_B.state IS standing:
            THEN TRANSFER momentum from pin_A to pin_B
            SET pin_B.state to falling

    IF all pins and ball have stopped moving:
      THEN SET game_state to frame_over
      SET initial_hit_type to none

  IF game_state IS frame_over:
    THEN SET game_state to reset_sequence
    SET sweep_arm.state to lowering

  IF sweep_arm.state IS lowering AND sweep_arm reaches lane:
    THEN SET sweep_arm.state to sweeping

  IF sweep_arm.state IS sweeping:
    MOVE sweep_arm forward
    FOR EACH pin in pins:
      IF sweep_arm collides_with pin:
        THEN REMOVE pin from lane
    IF sweep_arm reaches end of pin deck:
      THEN SET sweep_arm.state to retracting

  IF sweep_arm.state IS retracting AND sweep_arm is fully retracted:
    THEN SET pinsetter_deck.state to lowering

  IF game_state IS reset_sequence AND pinsetter_deck.state IS lowered:
    THEN SET pinsetter_deck.state to raising
    CREATE new set of 10 standing pins at default positions
    IF pinsetter_deck is fully raised:
      THEN SET game_state to ready_for_roll
```

Round 8  —  frames [21, 22, 23]
==================================================

```pseudocode
STATE VARIABLES:
  game_state: (ready, ball_rolling, pins_interacting, end_of_roll, clearing_fallen_pins, clearing_all_pins, setting_new_pins)
  roll_number: (1, 2)
  ball: Ball object with properties (position, velocity)
  pins: LIST of Pin objects, each with properties (id, position, state: {standing, falling, fallen})
  sweep_arm: SweepArm object with properties (state: {retracted, lowering, sweeping, retracting})
  pinsetter_deck: PinsetterDeck object with properties (state: {raised, lowering, lowered, raising})
  ui_text: STRING

RULES:
  IF game_state IS ready AND a ball is detected entering the lane:
    THEN SET game_state to ball_rolling
    CLEAR ui_text

  IF game_state IS ball_rolling AND ball collides_with a pin:
    THEN SET game_state to pins_interacting
    TRANSFER momentum from ball to the hit pin
    SET hit_pin.state to falling

  IF game_state IS pins_interacting:
    UPDATE ball.position and ball.velocity
    FOR EACH pin_A in pins:
      IF pin_A.state IS falling:
        UPDATE pin_A.position and pin_A.velocity
        FOR EACH pin_B in pins where pin_A IS NOT pin_B:
          IF pin_A collides_with pin_B AND pin_B.state IS standing:
            THEN TRANSFER momentum from pin_A to pin_B
            SET pin_B.state to falling
    IF all pins and ball have stopped moving:
      THEN SET game_state to end_of_roll

  IF game_state IS end_of_roll:
    LET standing_pins = GET_PINS_BY_STATE(standing)
    IF roll_number IS 1:
      IF COUNT(standing_pins) IS 0: // Strike
        THEN SET ui_text to "Strike"
        SET game_state to clearing_all_pins
      ELSE: // Not a strike
        IF standing_pins contains only pin_4 and pin_10:
          THEN SET ui_text to "4 - 10"
        // ... other split detections
        SET roll_number to 2
        SET game_state to clearing_fallen_pins
    ELSE IF roll_number IS 2:
      // ... Logic for Spare or Open Frame UI text
      SET game_state to clearing_all_pins
    DISPLAY ui_text

  IF game_state IS clearing_fallen_pins OR game_state IS clearing_all_pins:
    IF sweep_arm.state IS retracted:
      THEN SET sweep_arm.state to lowering
    IF sweep_arm.state IS lowering AND sweep_arm reaches lane:
      THEN SET sweep_arm.state to sweeping
    IF sweep_arm.state IS sweeping:
      MOVE sweep_arm forward
      FOR EACH pin in pins:
        IF game_state IS clearing_fallen_pins AND pin.state IS NOT standing:
          IF sweep_arm collides_with pin: THEN REMOVE pin
        ELSE IF game_state IS clearing_all_pins:
          IF sweep_arm collides_with pin: THEN REMOVE pin
      IF sweep_arm reaches end of pin deck:
        THEN SET sweep_arm.state to retracting
    IF sweep_arm.state IS retracting AND sweep_arm is fully retracted:
      IF game_state IS clearing_fallen_pins:
        THEN SET game_state to ready
      ELSE IF game_state IS clearing_all_pins:
        THEN SET game_state to setting_new_pins

  IF game_state IS setting_new_pins:
    IF pinsetter_deck.state IS raised:
      THEN SET pinsetter_deck.state to lowering
    IF pinsetter_deck.state IS lowering AND pinsetter_deck is fully lowered:
      THEN SET pinsetter_deck.state to raising
      CREATE new set of 10 standing pins at default positions
    IF pinsetter_deck.state IS raising AND pinsetter_deck is fully raised:
      THEN SET game_state to ready
      SET roll_number to 1
```

```pseudocode
  ball: Ball object with properties (position, velocity)
  pins: LIST of Pin objects, each with properties (id, position, state: {standing, falling, fallen, lifted})
  sweep_arm: SweepArm object with properties (state: {retracted, lowering, sweeping, retracting})
  pinsetter_deck: PinsetterDeck object with properties (state: {raised, lowering, on_deck, lifting, spotting, raising})
  ui_text: STRING

RULES:
  IF game_state IS ready AND a ball is detected entering the lane:
    THEN SET game_state to ball_rolling
    CLEAR ui_text

  IF game_state IS ball_rolling AND ball collides_with a pin:
    THEN SET game_state to pins_interacting
    TRANSFER momentum from ball to the hit pin
    SET hit_pin.state to falling

  IF game_state IS pins_interacting:
    UPDATE ball.position and ball.velocity
    FOR EACH pin_A in pins:
      IF pin_A.state IS falling:
        UPDATE pin_A.position and pin_A.velocity
        FOR EACH pin_B in pins where pin_A IS NOT pin_B:
          IF pin_A collides_with pin_B AND pin_B.state IS standing:
            THEN TRANSFER momentum from pin_A to pin_B
            SET pin_B.state to falling
    IF all pins and ball have stopped moving:
      THEN SET game_state to end_of_roll

  IF game_state IS end_of_roll AND reset_cycle IS idle:
    LET standing_pins = GET_PINS_BY_STATE(standing)
    IF roll_number IS 1:
      IF COUNT(standing_pins) IS 0: // Strike
        SET ui_text to "Strike"
        SET reset_cycle to resetting_frame
      ELSE: // Open first roll
        // ... Logic for split detection UI text
        SET reset_cycle to preparing_spare
    ELSE IF roll_number IS 2:
      // ... Logic for Spare or Open Frame UI text
      SET reset_cycle to resetting_frame
    DISPLAY ui_text

  // Spare preparation cycle
  IF reset_cycle IS preparing_spare:
    IF pinsetter_deck.state IS raised: THEN SET pinsetter_deck.state to lowering
    IF pinsetter_deck.state IS on_deck:
      FOR EACH pin in pins WHERE pin.state IS standing:
        SET pin.state to lifted
      SET pinsetter_deck.state to lifting
    IF pinsetter_deck.state IS lifting AND deck is clear of sweep path:
      SET sweep_arm.state to lowering
    IF sweep_arm.state IS lowering AND arm reaches lane: SET sweep_arm.state to sweeping
    IF sweep_arm.state IS sweeping:
      MOVE sweep_arm forward
      REMOVE all pins WHERE state IS fallen
      IF sweep_arm reaches end of deck: SET sweep_arm.state to retracting
    IF sweep_arm.state IS retracted AND pinsetter_deck.state IS lifting:
      SET pinsetter_deck.state to spotting
    IF pinsetter_deck.state IS spotting AND deck is on_deck:
      FOR EACH pin in pins WHERE pin.state IS lifted:
        SET pin.state to standing
      SET pinsetter_deck.state to raising
    IF pinsetter_deck.state IS raised:
      SET reset_cycle to idle
      SET game_state to ready
      SET roll_number to 2

  // New frame reset cycle
  IF reset_cycle IS resetting_frame:
    IF sweep_arm.state IS retracted: SET sweep_arm.state to lowering
    IF sweep_arm.state IS lowering AND arm reaches lane: SET sweep_arm.state to sweeping
    IF sweep_arm.state IS sweeping:
      MOVE sweep_arm forward
      REMOVE all pins from lane
      IF sweep_arm reaches end of deck: SET sweep_arm.state to retracting
    IF sweep_arm.state IS retracted AND pinsetter_deck.state IS raised:
      SET pinsetter_deck.state to lowering
    IF pinsetter_deck.state IS on_deck:
      CREATE new set of 10 standing pins
      SET pinsetter_deck.state to raising
    IF pinsetter_deck.state IS raised:
      SET reset_cycle to idle
      SET game_state to ready
      SET roll_number to 1
```

```pseudocode
STATE VARIABLES:
  simulation_state: (active, finished)
  game_state: (ready, ball_rolling, pins_interacting, end_of_roll)
  reset_cycle: (idle, preparing_spare, resetting_frame)
  roll_number: (1, 2)
  game_is_over: BOOLEAN
  initial_hit_type: (none, pocket, brooklyn, nose_hit)
  ball: Ball object with properties (position, velocity)
  pins: LIST of Pin objects, each with properties (id, position, state: {standing, falling, fallen, lifted})
  sweep_arm: SweepArm object with properties (state: {retracted, lowering, sweeping, retracting})
  pinsetter_deck: PinsetterDeck object with properties (state: {raised, lowering, on_deck, lifting, spotting, raising})
  ui_text: STRING

RULES:
  IF simulation_state IS finished:
    THEN RENDER black_screen
    // No other rules are processed

  ELSE IF simulation_state IS active:
    IF game_state IS ready AND a ball is detected entering the lane:
      THEN SET game_state to ball_rolling
      SET initial_hit_type to none
      CLEAR ui_text

    IF game_state IS ball_rolling AND ball collides_with a pin for the first time:
      THEN SET game_state to pins_interacting
      IF first pin hit is pin_1 AND ball.impact_point is on the left side of pin_1:
        THEN SET initial_hit_type to brooklyn
      ELSE IF first pin hit is pin_1 AND ball.impact_point is on the right side of pin_1:
        THEN SET initial_hit_type to pocket
      ELSE IF first pin hit is pin_1 AND ball.impact_point is near the center of pin_1:
        THEN SET initial_hit_type to nose_hit
      TRANSFER momentum from ball to the hit pin(s)
      SET hit_pin(s).state to falling

    IF game_state IS pins_interacting:
      UPDATE ball.position and ball.velocity
      FOR EACH pin_A in pins:
        IF pin_A.state IS falling:
          UPDATE pin_A.position and pin_A.velocity
          FOR EACH pin_B in pins where pin_A IS NOT pin_B:
            IF pin_A collides_with pin_B AND pin_B.state IS standing:
              THEN TRANSFER momentum from pin_A to pin_B
              SET pin_B.state to falling
      IF all pins and ball have stopped moving:
        THEN SET game_state to end_of_roll

    IF game_state IS end_of_roll AND reset_cycle IS idle:
      LET standing_pins = GET_PINS_BY_STATE(standing)
      IF roll_number IS 1:
        IF COUNT(standing_pins) IS 0: // Strike
          IF initial_hit_type IS brooklyn: THEN SET ui_text to "Brooklyn Strike"
          ELSE: THEN SET ui_text to "Strike"
          SET reset_cycle to resetting_frame
        ELSE: // Open first roll
          SET reset_cycle to preparing_spare
      ELSE IF roll_number IS 2:
        SET reset_cycle to resetting_frame
      IF game_completion_condition is met: // e.g., 10th frame is over
        SET game_is_over to TRUE
      DISPLAY ui_text

    IF reset_cycle IS preparing_spare:
      IF pinsetter_deck.state IS raised: THEN SET pinsetter_deck.state to lowering
      IF pinsetter_deck.state IS on_deck:
        FOR EACH pin in pins WHERE pin.state IS standing: SET pin.state to lifted
        SET pinsetter_deck.state to lifting
      IF pinsetter_deck.state IS lifting AND deck is clear of sweep path: SET sweep_arm.state to lowering
      IF sweep_arm.state IS lowering AND arm reaches lane: SET sweep_arm.state to sweeping
      IF sweep_arm.state IS sweeping:
        MOVE sweep_arm forward; REMOVE all pins WHERE state IS fallen
        IF sweep_arm reaches end of deck: SET sweep_arm.state to retracting
      IF sweep_arm.state IS retracted AND pinsetter_deck.state IS lifting: SET pinsetter_deck.state to spotting
      IF pinsetter_deck.state IS spotting AND deck is on_deck:
        FOR EACH pin in pins WHERE pin.state IS lifted: SET pin.state to standing
        SET pinsetter_deck.state to raising
      IF pinsetter_deck.state IS raised:
        SET reset_cycle to idle; SET game_state to ready; SET roll_number to 2

    IF reset_cycle IS resetting_frame:
      IF sweep_arm.state IS retracted: SET sweep_arm.state to lowering
      IF sweep_arm.state IS lowering AND arm reaches lane: SET sweep_arm.state to sweeping
      IF sweep_arm.state IS sweeping:
        MOVE sweep_arm forward; REMOVE all pins from lane
        IF sweep_arm reaches end of deck: SET sweep_arm.state to retracting
      IF sweep_arm.state IS retracted AND pinsetter_deck.state IS raised: SET pinsetter_deck.state to lowering
      IF pinsetter_deck.state IS on_deck:
        CREATE new set of 10 standing pins
        SET pinsetter_deck.state to raising
      IF pinsetter_deck.state IS raised:
        IF game_is_over IS TRUE:
          THEN SET simulation_state to finished
        ELSE:
          SET reset_cycle to idle; SET game_state to ready; SET roll_number to 1
```
